* Mejoras que se pueden aplicar:

	- Modelo el genero debe ser un Enum.
	- Deberia usar DI para poder hacer test del controller, por ejemplo.
	- Tambien podría crear test de UI con Selenium (en mi github podeis ver ejemplos)
	- Test de JS.
	- En el front, se puede mejorar introduciendo validaciones antes de enviar al Back:
		Que al menos tenga un valor.
		Que se corresponda con el modelo.
		etc.
